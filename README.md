# 符寶網 - 全球玄門文化科普交易平台

## 技術棧

| 層級 | 技術 |
|------|------|
| 前端框架 | Next.js 16 (App Router) |
| UI組件 | React 19 + shadcn/ui |
| 樣式 | TypeScript 5 + Tailwind CSS 4 |
| 後端 | PHP 7.4+ (ThinkPHP 風格) |
| 數據庫 | MySQL 5.7+ |
| 認證 | JWT |

## 快速開始

```bash
# 安裝依賴
pnpm install

# 開發模式 (端口 5000)
pnpm dev

# 生產構建
pnpm build
```
